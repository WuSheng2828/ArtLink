# ArtLink - ComfyUI QQ 机器人

通过 QQ（NapCat）远程控制 ComfyUI 进行 AI 绘图和图片反推。

---

## 功能

### 图片生成
- 发送提示词即可生图，支持多种尺寸
- 从 ComfyUI 获取图片，无需配置输出目录
- 每晚 0 点自动刷新每日次数

### 文字反推
- 发送图片，自动调用 WD14 Tagger 反推提示词
- 图片下载失败自动重试，无需担心网络问题

### 任务撤回
- 群内直接发送“撤回”，两分钟内生成的任务可撤回
- 成功无反馈，失败自动提示并撤回

### 多图工作流
- 同时生成多张图片，按顺序发送，带序号
- 发送间隔 1 秒，防止刷屏

### 尺寸记忆
- 每位用户独立记忆尺寸设置
- 后台可自定义任意尺寸及触发词

### 提示词过滤
- 可设置过滤词，自动去除敏感或无用词汇

### 管理员菜单
- 通过 QQ 私聊管理群/私聊白名单、黑名单
- 支持快捷指令，无需打开后台

### 其他
- 排队提示、失败重试、生成耗时显示
- 群聊神秘返图、私聊详细返图
- Web 后台可视化配置，支持历史记录查询

## 部署

### 环境要求
- Windows 10/11
- NapCat QQ 机器人（HTTP:3000, WS:3001）
- ComfyUI（默认 http://127.0.0.1:8188）
- Python 3.12（推荐使用便携版，也可用系统环境）

### 安装步骤

#### 方式一：使用便携 Python 环境（推荐，无需安装 Python）

1. **下载便携 Python 包**
   - 前往 [Releases 页面](https://github.com/WuSheng2828/ArtLink/releases)
   - 下载 `python311.zip`（由作者打包好，解压直接用）

2. **解压便携包**
   - 将 `python311.zip` 解压到本项目根目录
   - 解压后应看到 `Python311/python.exe`

#### 方式二：自己配置 Python 环境

1. **下载 Python 便携包**
   - 前往 [Python 3.12 下载页面](https://www.python.org/downloads/release/python-3120/)
   - 下载 `Windows embeddable package (64-bit)`

2. **解压并配置**
   - 解压到项目根目录的 `Python311` 文件夹
   - 编辑 `Python311/python312._pth`，在文件末尾添加一行 `Lib\site-packages` 并保存

## 使用

### 群聊指令（需 @机器人）
- `绘图 提示词`：生成图片
- `反推 + 图片`：获取图片标签
- `正方` / `横版` / `竖版`：切换尺寸
- `注册`：注册成为用户
- `说明`：查看帮助
- `撤回`：撤回最近生成的图片（无需 @）

### 私聊指令
- 同上（需在私聊白名单中）

### 管理员指令
- `管理`：进入管理菜单
- `状态`：查看系统状态
- `过滤开` / `过滤关`：控制过滤
