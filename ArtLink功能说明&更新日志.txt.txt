                    一、系统概览
═══════════════════════════════════════════════════════════════

ArtLink 是一个单文件 Python 脚本，连接 QQ（NapCat）与 ComfyUI，
实现通过 QQ 私聊或群聊远程控制 ComfyUI 进行 AI 绘图和图片反推。

【核心链路】
  QQ 消息 → NapCat WebSocket → ArtLink → ComfyUI API
                    ↓
  生成完成 ← NapCat HTTP ← ArtLink ← ComfyUI /view 接口

【技术栈】
  Python 3.12 · Flask · SQLite · WebSocket · ComfyUI API


═══════════════════════════════════════════════════════════════
                    二、部署要求
═══════════════════════════════════════════════════════════════

【环境】
  Windows 10/11
  Python 3.12（支持便携版或系统环境，启动脚本自动检测）
  NapCat QQ 机器人 (HTTP:3000, WS:3001)
  ComfyUI (端口 8188)

【依赖】
  pip install flask requests websocket-client
  便携环境缺少依赖时自动安装，系统环境缺少依赖时给出安装提示。

【启动】
  双击 run.bat → 自动检测 Python 环境 → 启动服务 → 自动打开浏览器 → http://127.0.0.1:5000

                    三、核心功能
═══════════════════════════════════════════════════════════════

✅ 图片生成
   - 通过 QQ 发送提示词，调用 ComfyUI 工作流生成图片
   - 支持动态自定义尺寸（后台可任意添加/删除/修改尺寸及触发词）
   - 通过 /view 接口直接获取图片，无需配置 ComfyUI 输出目录
   - 生成完成后自动发送图片到 QQ（支持文字+图片混合显示）
   - 文生图消耗每日次数，每日 0 点自动刷新

✅ 文字反推
   - 发送图片给机器人，自动调用 WD14 Tagger 反推提示词
   - 反推结果以文字形式返回 QQ
   - 图片自动下载到 ComfyUI input 目录（后台可配置）
   - 图片下载失败会提示"❌ 图片下载失败，请稍后重试"
   - 反推功能不消耗每日次数

✅ 视频生成（v2.0.0 新增，v2.1.0 完善）
   - 支持 ComfyUI 视频工作流，通过 QQ 提交后生成视频
   - 视频文字说明和视频文件分两条消息分别发送，确保正常显示
   - 视频时长支持：后台配置「时长」整数节点，QQ 端三重询问流程
     ① 选择分辨率 → ② 输入时长（秒） → ③ 提交任务
   - 视频宽高分离：支持独立的「宽度(视频)」和「高度(视频)」整数节点
   - 兼容文生视频场景：有 Latent 节点的视频工作流，宽高也填入 Latent
   - 视频尺寸独立管理：后台"视频自定义尺寸"区域，与图片尺寸分离

✅ 多图/多提示词支持（v2.0.0 新增）
   - 多段提示词：按回车换行分割，按顺序填入多个提示词节点
   - 多张图片：按 QQ 发送顺序读取，按顺序填入多个图片节点
   - 数量不匹配时自动弹出二次确认（60 秒超时自动取消）

✅ 任务撤回
   - 群聊内直接发送"撤回"（无需 @机器人），可撤回自己 2 分钟内最新生成的任务
   - 成功撤回：无任何提示，静默完成
   - 撤回失败（超时）：提示"❌ 已超过2分钟，无法撤回"，该提示 15 秒后自动撤回
   - 不干扰其他成员的任务，仅限本人操作

✅ 多图工作流
   - 支持一次生成多张图片的工作流
   - 每条消息附带序号（第1/N张、第2/N张…）
   - 发送间隔 1 秒，避免刷屏

✅ 尺寸记忆
   - 不局限正方/横版/竖版，支持后台自定义增加任意尺寸和对应触发词
   - 每位用户独立记忆当前尺寸设置
   - 发送对应的尺寸触发词即可切换，下次生图沿用
   - 图片尺寸与视频尺寸独立管理

✅ 再来一张（v1.1.0 新增，v2.1.0 优化）
   - 用上一次生图的提示词再生成，无需重新输入
   - 私聊支持 1~5 张（发送「再来3张」）
   - 群聊支持 1 张（无需 @机器人）
   - 可自定义触发词（如：再来、继续、加图、再来一次）
   - 仅追溯最近一次图片任务，文字反推和视频任务不进入追溯
   - 提示词复用上次，尺寸取当前用户最新设置

✅ 绘图模式（v1.1.0 新增）
   - 私聊免触发词连续生图，发送提示词即出图
   - 进入词和超时分钟可在后台配置
   - 每次操作刷新计时，超时自动退出
   - 发送「退出」手动退出

✅ 随机提示词生图（v1.2.0 新增，v1.3.0 重做，v1.4.1 完善）
   - 私聊/群聊发送"随便来点"触发随机提示词生图
   - 36 分类词库（10 大类），标签码体系（~75 个标签码）
   - 支持标签关系引擎：对立（冲突重抽）+ 吸引（加权选择）
   - 每条关系可独立开关，强度分弱/中/强 3 档
   - 可视化词库编辑器：词条增删改、标签码管理、关系配置、取词数量、分类排序
   - 编辑器支持全局字号调节，响应式布局

✅ 排队提示
   - 多人同时生图时显示"前边还有N个任务"
   - 任务完成后自动更新排队计数

✅ 失败自动重试
   - 生成超时自动重试 1 次
   - 两次都失败才通知用户

✅ 生成耗时
   - 任务完成时显示耗时（如"23秒"）
   - 历史记录中也保存耗时数据

✅ 提示词过滤（v1.1.0 新增）
   - 后台可设置过滤开关和过滤词列表（逗号分隔）
   - 开启后自动从提示词中移除过滤词（不区分大小写，精确匹配单词边界）
   - 管理员可通过私聊"过滤开"/"过滤关"快捷指令控制
   - 过滤词预设功能：可添加多组预设，点"应用"追加进过滤池（不覆盖）

✅ 消息自动撤回
   - 群聊提交消息 15 秒后自动撤回
   - 私聊提交消息同样 15 秒后撤回
   - 尺寸切换提示在群聊中 15 秒后撤回
   - 管理菜单消息在操作后自动撤回上一条

✅ 群聊神秘返图
   - 群聊中返图不 @ 任何人，不显示提示词
   - 只显示"✨ 任务完成 · 23秒" + 图片

✅ 私聊完整返图
   - 私聊中返图保留完整信息：提示词、尺寸、耗时

✅ 返图自定义显示内容
   - 私聊/群聊各自独立勾选（提示词/尺寸/耗时/工作流名）
   - 私聊默认全部显示，群聊默认仅显示耗时
   - 「随便来点」有独立显示设置

✅ ComfyUI 在线检测
   - 被动检测：每次提交任务时检测（秒级感知）
   - 主动检测：每 30 秒心跳一次，连续 3 次失败判定离线
   - 离线/恢复时自动通知所有管理员


═══════════════════════════════════════════════════════════════
                    四、权限体系
═══════════════════════════════════════════════════════════════

【多管理员】
  不再使用单一 admin_qq，改为独立管理员表（admin_list），可在后台任意添加/删除管理员。
  所有管理员拥有同等权限（快捷指令、管理菜单、状态查询等）。

【六层白名单】
  第一层：多管理员       → 管理菜单、状态查看、快捷指令、过滤开关
  第二层：白名单总开关   → 私聊/群聊可独立关闭（关闭后全员可用或全群静默）
  第三层：群白名单       → 只有已添加的群才能交互，每群可独立开关
  第四层：群成员白名单   → 群内白名单成员可用，支持自助注册
  第五层：私聊白名单     → 控制私聊使用权限，未授权每 30 秒才提醒一次
  第六层：黑名单        → 拉黑后无法注册也不响应任何消息

【白名单开关（v2.0.0 新增）】
  私聊白名单总开关：后台系统配置中可开启/关闭，关闭时任何 QQ 号都可私聊
  群聊白名单总开关：后台系统配置中可开启/关闭，关闭时所有群消息静默
  群聊独立小开关：每个群可在权限管理中单独启停，即时生效
  白名单权限缓存：首次检查后缓存，后续秒过，配置变更即时刷新

【未注册用户提示】
  在群聊中，未加入群成员白名单的成员 @机器人，除"注册"外，回复提示"请先注册"（15 秒后自动撤回）。

【私聊无权限静默期】
  非白名单用户私聊机器人时，首次提示"无权限"，30 秒内重复私聊不再提示。

【自助注册】
  群内发送: @机器人 注册
  自动加入当前群的成员白名单，备注自动填群昵称
  若已注册再次发送"注册"，提示"已在白名单"并 15 秒后自动撤回

【管理员权限】
  所有管理员 QQ（后台设置）均可私聊：
    - 管理  → 进入交互式管理菜单（群聊/私聊/其他设置）
    - 状态  → 查看系统运行状态
    - 快捷指令 → 重置群/拉黑/移除/添加群/添加私聊/移除私聊（见指令部分）

【每日次数限制】
   - 每用户每天生图次数（仅限制群聊，私聊无限制）
   - 每日 0 点自动刷新
   - 支持工作流单独限次：① 跟随总上限  ② 单独 N 次/天  ③ 无限制（-1）
   - 群聊提交时按工作流模式显示不同次数提醒
   - 管理员可在后台或管理菜单中重置指定成员或整群次数

                    五、工作流管理
═══════════════════════════════════════════════════════════════

✅ 多工作流支持
   - 可在后台添加/删除/编辑多个工作流
   - 每个工作流可设置：名称、JSON文件、输出类型(图片/文字/视频)、备注、示例

✅ 弹窗式工作流管理（v2.0.0 重做，v2.1.0 完善）
   - 点击"添加工作流"弹出独立配置窗口，支持添加和编辑双模式
   - 可视化选择 JSON 文件（自动扫描项目目录）
   - 动态节点管理：可添加任意数量的节点
   - 旧工作流节点字段自动兼容

✅ 6 种节点类型（v2.1.0 完善）

  ┌─────────────┬──────────────────────────────────────┐
  │ 节点类型    │ 说明                                  │
  ├─────────────┼──────────────────────────────────────┤
  │ 提示词      │ 正面提示词文本输入（支持复数，按序填入）│
  │ Latent      │ 图像潜在空间节点（宽×高，支持复数）    │
  │ 图片        │ 图片输入（用于图生图/反推，支持复数）   │
  │ 宽度(视频)  │ 视频宽度整数节点                      │
  │ 高度(视频)  │ 视频高度整数节点                      │
  │ 时长(视频)  │ 视频时长整数节点（秒）                 │
  └─────────────┴──────────────────────────────────────┘

  每个节点通过「编号」（纯数字匹配节点 ID）或「标题」（文字匹配 _meta.title）定位。
  节点数量不匹配时自动弹出二次确认。

✅ 触发词 & 用量合并管理（v2.1.0 优化）
   - 每个触发词对应一个工作流，以彩色徽章卡片形式展示
   - 卡片上直接显示并切换用量模式（跟随上限/单独N次/无限制），即改即存
   - 新增触发词时一步设置用量模式
   - 支持删除触发词

✅ 工作流卡片展示（v2.1.0 新增）
   - 响应式 Grid 布局：大屏 3 列 / 中等 2 列 / 小屏 1 列
   - 每张卡片展示：名称 + 类型标签 | 文件路径 | 节点彩色标签 | 备注 | 示例
   - 支持在卡片上直接编辑已有工作流（弹窗预填数据）

✅ 图片/视频尺寸
   - 各自独立管理，支持动态添加/删除/修改
   - 每个尺寸有名称、宽、高、触发词
   - 视频尺寸提交后询问用户选择分辨率

✅ 工作流使用限制（v1.1.0 新增，v2.1.0 与触发词合并）
   - 三种模式：跟随总上限 / 单独N次 / 无限制
   - 群聊提交时按工作流模式显示对应的次数提醒
     「跟随上限」显示全局剩余，「单独上限」显示工作流剩余X/N，「无限制」不显示

                    六、QQ 交互指令
═══════════════════════════════════════════════════════════════

【群聊指令】(需 @机器人，但"撤回"和"再来"不强制 @)
  绘图 + 提示词       → 触发生图（消耗次数，每日0点刷新）
  反推 + 图片         → 触发图片反推（不消耗次数）
  正方 / 横版 / 竖版  → 切换预设尺寸（15 秒后自动撤回切换提示，支持后台自定义更多尺寸触发词）
  注册                → 自助加入白名单（已注册则提示 15 秒后撤回）
  说明                → 查看帮助（可配置多个触发词，如 帮助/help/?等）
  撤回                → 撤回自己 2 分钟内最新生成的任务（直接发送"撤回"二字，无需 @机器人）
  再来                → 用上次提示词再生成 1 张（无需 @，仅追溯图片任务，独立开关控制）
  随便来点            → 随机提示词生图（需 @机器人，后台可开启/关闭）

【私聊指令】
  绘图 + 提示词       → 触发生图 (需在私聊白名单)
  反推 + 图片         → 触发图片反推 (需在私聊白名单)
  正方 / 横版 / 竖版  → 切换尺寸（支持后台自定义的触发词）
  说明                → 查看帮助（可配置多个触发词）
  再来 + 数字         → 用上次提示词再生成（支持 1~5 张及中文一二三四五两，仅追溯图片任务，独立开关）
  绘图模式            → 进入免触发词生图模式（超时自动退出）
  随便来点            → 随机提示词生图（直接发送，后台可开启/关闭）

【管理员私聊指令】
  管理                → 进入管理菜单（群聊/私聊/其他设置，支持确认简化，30 秒无操作自动退出）
  状态                → 查看系统运行状态
  过滤开 / 过滤关     → 控制提示词过滤开关
  快捷指令：
    重置群 群号         → 重置该群所有成员次数
    拉黑 群号 QQ号      → 将指定群成员拉黑
    移除 群号 QQ号      → 将指定群成员移出白名单
    重置 群号 QQ号      → 重置该成员今日次数
    添加群 群号         → 将群加入白名单
    添加私聊 QQ号 备注  → 添加私聊白名单
    移除私聊 QQ号       → 移除私聊白名单

【指令格式】
  触发词 + 内容
  示例: 绘图 1girl, solo, masterpiece
  示例: 竖版 绘图 1girl, solo
  示例: 反推 + [发送一张图片]
  示例: 撤回（直接发送，无需 @）
  示例: 再来3张（私聊）/ 再来（群聊）
  示例: 随便来点（直接发送或 @机器人 + 随便来点）

                    七、Web 管理后台
═══════════════════════════════════════════════════════════════

【地址】http://127.0.0.1:5000

【选项卡 1：系统配置】（v2.1.0 重新设计）

  以分组卡片形式展示，每组有功能简介：

  ┌─ 连接配置 ── 通讯地址管理
  │   ComfyUI输入目录 / NapCat API / ComfyUI 地址
  ├─ 使用控制 ── 限制使用频率与访问权限
  │   每日上限（仅限制群聊，私聊无限制）/ 私聊+群聊白名单开关
  ├─ 帮助说明 ── 用户发 ？/帮助/help 时看到的内容
  │   公共说明（所有人可见）/ 管理员说明（管理员看到管理版，留空用系统默认）
  │   触发词设置（多个用逗号分隔，管理员与普通用户共用触发词但内容不同）
  ├─ 再来一张 ── 用上次提示词再生成（仅追溯图片任务）
  │   私聊群聊独立开关 / 触发词自定义
  ├─ 绘图模式 ── 私聊免触发词连续生图
  │   进入词 / 超时分钟
  └─ 返回内容 ── 控制返图附带哪些信息
      提示词·尺寸·耗时·工作流名（私聊/群聊独立勾选）

【选项卡 2：权限管理】

  👑 管理员管理
    - 添加/删除管理员 QQ，支持备注

  👤 私聊白名单
    - 添加/删除 QQ 号，支持备注

  👥 群聊白名单
    - 添加/删除群，每群可独立开启/关闭
    - 删除群时自动清理成员和黑名单

  👥 群成员白名单 & 次数管理
    - 选择群 → 查看成员列表 + 今日已用次数
    - 添加/删除成员、重置单个成员次数、重置整群次数、拉黑成员

  🚫 黑名单管理
    - 选择群 → 查看黑名单，可取消拉黑（恢复白名单）

                    八、词库系统（v1.3.0+ 新增）
═══════════════════════════════════════════════════════════════

ArtLink v1.3.0 起，随机词库从旧的 27 分类 type 字段体系，升级为 36 分类 +
标签码体系 + 标签关系引擎。

【标签码系统】
  每条词携带多个标签码（tags 数组），替代旧的 sfw/nsfw type：
    1girl → [sfw, 女性, 人类]
    fox ears → [sfw, 兽耳, 狐耳]

  标签码 ~75 个，分 9 个维度：等级/性别/年龄/种族/身体/服装/性内容/场景/风格

【36 分类结构】
  十大类：基础信息(1-3)、外貌特征(4-7)、身体特征(8-12)、服装穿戴(13-16)、
          道具物品(17-18)、姿态动作(19-23)、性感暴露(24-27)、场景背景(28-31)、
          镜头构图(32-34)、艺术交互(35-36)

【标签关系引擎】
  对立关系 — 两标签同时出现时触发重抽（强度：弱/中/强）
  吸引关系 — 前面选了某标签后，后续相关标签在随机时获得权重加成
              （强×3、中×2、弱×1.5）
  每条关系有独立 enable 开关，可随时启停。

【词库可视化编辑器（端口 19876）】
  三个选项卡：
    词库管理 — 集中添加区 + 36 分类卡片网格 + 拖拽 + 临时池
    规则管理 — 规则开关 + 标签码管理 + 标签关系（独立开关/编辑/删除）
    随机设置 — 取词数量紧凑网格 + 分类排序拖拽
  字号滑块 A- / A+（自动记忆）

【词库分块管理】
  分块文件放 wordbank_parts_v2/，双击 merge_wordbank_v2.bat 增量合并。
  支持 pick_min:sfw / pick_max:nsfw 不同取值格式。
  跳过含"指令""说明"的非词库文件。

【配置文件】
  wordbank_filter_config.json — 规则配置：
    category_order / category_order_list — 分类顺序
    tag_relations_enabled — 关系引擎总开关
    tag_relations — 关系列表（含 enabled 独立开关）
    all_tags — 标签码全集

    外置 wordbank.json 词库（26分类，1500+条目）
    26个分类按 Anima 模板槽位顺序排列
    后台独立选项卡管理：总开关/触发词/群私聊开关/SFW+NSFW勾选
    词库管理：添加词条（选分类+标签文本+类型）+ 搜索去重 + 刷新
    SFW 模式仅从 sfw 词条生成，NSFW 模式仅从 nsfw 词条生成
    两个都勾选则混合随机
  + 词库分块管理系统
    新增 wordbank_parts/ 文件夹，支持 .txt 和 .json 分块文件
    新增 merge_wordbank.bat + merge_wordbank.py 合并去重脚本
    .txt 文件简洁格式：#分类ID + 每行标签,sfw/nsfw
    支持中文分类名映射（#发色 → 分类2，#场景 → 分类23 等）
    合并时自动去重（忽略大小写），按文件名排序
  + 再来一张改用数据库回退
    优先读内存缓存，没有则从 history 表查最后一次成功生图记录
    重启 ArtLink 后依然可用（群聊+私聊均支持）
  + 尺寸优化：再来一张时尺寸传 None，自动使用用户最新尺寸记忆
  + 后台选项卡架构改造
    系统配置和随便来点使用选项卡切换，布局更清晰
    所有原功能完全保留，新增随便来点独立选项卡
  + 状态查询增加随便来点开关状态
  + 帮助文本增加随便来点说明
  + 随机词库系统完善
    词库扩充至 2260+ 条，完善 26 分类体系
    优化提示词组合逻辑，提升生成内容的丰富度
  + 词库分块管理工具优化
    merge_wordbank.bat 支持便携 Python 环境检测
    merge_wordbank.bat 同时支持 .json 和 .txt 文件
    词库文件可拆分到 wordbank_parts/ 文件夹管理
  + 版本号 + 文件头更新为 v1.2.0

v1.1.0 (2026-05-28)
  + 工作流单独限次：后台独立区域，仅显示已绑定触发词的工作流
    支持"跟随总上限 / 单独上限N次 / 无限制(-1)"三种模式
    通过下拉选择器直接修改，无需弹窗，仅影响群聊
  + 群聊「再来一张」：群内直接发送"再来"（无需 @机器人）
    仅支持 1 张，独立开关控制，互不干扰
    记录独立于私聊（不同群/不同人独立记忆）
  + 私聊「再来一张」优化：支持中文"两"（再来两张=2张）
    多张生成完毕后追加提示"✅ X张全部生成完毕"
    独立开关控制
  + 私聊「绘图模式」优化：进入后输入尺寸触发词优先切换尺寸（不生图）
    仅识别完全匹配的尺寸触发词，避免误判
  + 返图自定义显示内容：私聊/群聊各自独立勾选（提示词/尺寸/耗时/工作流名）
    私聊默认全部显示，群聊默认仅显示耗时
  + 后台"再来一张"设置：私聊和群聊各自独立开关
    管理菜单「其他设置」新增「5. 再来一张设置」子菜单
  + 历史记录合并：一条任务只产生一条历史记录
    提交时创建，完成后 UPDATE 状态，不再新增多余记录
    历史表格新增"类型"列，正确区分私聊/群聊
  + 排队文案优化：从"排队N人"改为"前边还有N个任务"
  + 控制台日志增强：显示用户QQ、私聊/群聊、工作流、结果，不显示提示词
  + 选项卡折叠记忆：刷新页面后自动恢复展开/收起状态
  + 数据库新增表：filter_presets（过滤词预设）
  + 数据库新增表：workflow_daily_usage（工作流单独计数）
  + 新增过滤词预设功能：可添加多组预设，点"应用"追加进过滤池（不覆盖）
  + 提示词过滤独立区域：大文本框+折叠三角，管理更清晰
  + 四个名单全部可折叠：私聊白名单/群聊白名单/群成员白名单/黑名单
